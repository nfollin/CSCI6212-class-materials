## Merge Sort

    MergeSort (array a, size n)
        if n == 1
            return a
        else
            a1 = MergeSort(a[1 .. n/2], n/2)
            a2 = MergeSort(a[n/1 + 1 .. n], n/2)
            return Merge(a1, a2, n)


    Merge (array a1, array a2, size n)
        result = empty array of size n
        p1 = 1
        p2 = 1
        for n times:
            if a1[p1] < a2[p2]
                append a1[p1] to result
                p1++
            else
                append a2[p2] to result
                p2++
        return result
